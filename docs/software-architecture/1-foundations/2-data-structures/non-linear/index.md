**Practical examples:**

- BST for a system to quickly search for users by ID
- Heap for a priority queue
- Trie for autocomplete
- Graph for a social network (friends, connections)
- Dijkstra's algorithm to find the fastest route on maps

**Real-world use cases:**

- Search engine: Trie for autocomplete
- Database: B-Tree for indexing
- Social network: Graph to find mutual friends (BFS)
- Routing system: Dijkstra's algorithm for GPS/maps
- Module dependencies: topological sort for compilation order
