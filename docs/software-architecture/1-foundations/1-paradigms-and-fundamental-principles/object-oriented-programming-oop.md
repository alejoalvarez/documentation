# Object Oriented Programming (OOP)

**Topics to master:**

- Classes and objects
- Encapsulation (access modifiers)
- Inheritance and polymorphism
- Abstraction
- Composition vs. inheritance
- Abstract methods and interfaces
- Exceptions and error handling

**Recommended resources:**

- Book: “Effective Java” by Joshua Bloch (3rd edition)
- Course: “Object-Oriented Programming in Java” on Coursera
- Official Java/Python documentation
- Tutorial: Head First Design Patterns

**Practical examples:**

- Create a class hierarchy for a banking system (Account, SavingsAccount, CheckingAccount)
- Implement interfaces for different types of payments (Card, PayPal, Transfer)
- Design a permission system with proper encapsulation

**Real-world use case:**

- Employee management system: different types of employees (Executive, Contractor, Intern) with different benefits and responsibilities
- Implement polymorphism to calculate salaries according to employee type

**Exercises:**

1. Implement a library system with OOP (Book, User, Loan)
2. Create a game with inheritance (Character > Warrior, Mage, Archer)
3. Design an e-commerce site with composition (Cart, Product, Discount)
