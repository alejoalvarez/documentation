# SQL

An ```SQL``` (Structured Query Language) is a language used to manage and manipulate relational databases.

## Topics
- SELECT, INSERT, UPDATE, DELETE (CRUD)
- WHERE, ORDER BY, GROUP BY, HAVING
- Joins (INNER, LEFT, RIGHT, FULL, CROSS)
- Subqueries
- Aggregation (COUNT, SUM, AVG, MIN, MAX)
- String functions, Date functions, Math functions
- UNION, UNION ALL, INTERSECT, EXCEPT
- Transactions (COMMIT, ROLLBACK)

**Recommended resources:**

- Website: SQLZoo.net (interactive tutorial)

- User behavior analysis
