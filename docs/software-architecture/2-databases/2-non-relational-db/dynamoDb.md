# Dynamo DB

- Key-value and document database
- Primary keys (partition key and sort key)
- Secondary indexes (global and local)
- Read and write capacity units
- Transactions
- Streams
- Backup and restore
- Security

**Recommended resources:**

- DynamoDB documentation
- Book: “DynamoDB in Action” by Alex DeBrie

**Real-world use cases:**

- Storing flexible user data
- Blog system with nested comments
- Mobile app with hierarchical data
