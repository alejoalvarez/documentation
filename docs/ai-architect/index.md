# 🏗️ Ruta AI Software Architect

<span className="badge-arch">AI Architect</span>

Guía técnica para diseñar sistemas AI escalables, mantenibles y seguros. El AI Architect toma decisiones estratégicas que impactan cómo los modelos se integran en productos reales.

## ¿Qué hace un AI Software Architect?

A diferencia del AI Engineer (que implementa), el Architect **diseña el sistema**: define cómo los componentes se conectan, qué tecnologías usar y cómo garantizar escalabilidad, seguridad y gobernanza.

## Fases de la Ruta

| Fase | Duración | Temas |
|------|----------|-------|
| **Prerrequisito** | Variable | Clean Arch, Microservicios, Cloud, Seguridad |
| **Fase 2** | 2–4 meses | Comprensión de modelos, Data Arch, MLOps Platform |
| **Fase 3** | 3–5 meses | RAG Arch, Multi-Agent, LLM Gateway, Vector DBs |
| **Fase 4** | 2–3 meses | GPU Infra, Inference Optimization, Scalability |
| **Fase 5** | Continuo | AI Governance, AI Security, FinOps, Leadership |

## Punto de partida recomendado

Empieza por [Clean Architecture](./clean-architecture) — es el fundamento que unifica todo lo demás.
